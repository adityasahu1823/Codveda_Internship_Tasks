Welcome to my **Codveda Technology Internship** project repository!

##  Task Overview

###  Task 1 – Basic Calculator
A simple Java console application that performs:
- Addition  
- Subtraction  
- Multiplication  
- Division  

**Features:**
- Handles invalid operators  
- Checks for division by zero  

**File:** `Level1_Task1_BasicCalculator.java`

---

###  Task 2 – Number Guessing Game
A fun console game where:
- The program generates a random number between 1 and 100.  
- The user has limited attempts to guess it.  
- Provides feedback (“Too high” / “Too low”).  

**File:** `Level1_Task2_NumberGuessingGame.java`

---

###  Task 3 – Factorial Calculation (Recursion)
A program to calculate the factorial of a number using recursion.

**Features:**
- Handles negative numbers and zero.  
- Demonstrates basic recursion concept.  

**File:** `Level1_Task3_FactorialRecursion.java`
